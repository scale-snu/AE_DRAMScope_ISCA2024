# Retention time test

## Run

```
$ make
$ sudo ./run.sh
```

## Instructions

```
sudo ./Retention [-start start] [-bank bank] [-num_rows num_rows] [-sleep sleep]
          -start start               : start row address
          -bank bank                 : bank address
          -num_rows num_rows         : # of rows
          -sleep sleep               : retention time [us]
```
