# Find subarray sizes

## Run

```
$ make
$ sudo ./run.sh
```

## Instructions

```
$ sudo ./Rowcopy [-wait wait] [-bank bank] [-vendor vendor]
          -wait wait               : PRE to ACT time interval in rowcopy operation
          -bank bank               : bank address
          -vendor vendor           : DRAM vendor (e.g., s, h, m)
```
